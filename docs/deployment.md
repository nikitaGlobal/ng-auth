# Развёртывание

## Требования

- WordPress 6.0+
- PHP 8.0+
- WooCommerce 8.0+ (опционально)

## Установка

### Через WordPress Admin

1. Загрузите ZIP-архив плагина через «Плагины → Добавить новый → Загрузить плагин»
2. Активируйте плагин
3. Настройте через «NG Auth → Настройки»

### Вручную

1. Скопируйте папку `ng-auth` в `/wp-content/plugins/`
2. Активируйте плагин
3. Настройте через «NG Auth → Настройки»

### WP-CLI

```bash
wp plugin install /path/to/ng-auth.zip --activate
```

## Первоначальная настройка

1. Включите обязательное подтверждение: `NG Auth → Настройки → Общие`
2. Настройте роли и доменные зоны (опционально)
3. Настройте хотя бы одного провайдера (SMS или ЕСИА)
4. Для SMS-провайдера укажите API-ключи

## Массовая пометка существующих пользователей

```bash
wp ng-auth mark-unverified
```

## Активация рассылки напоминаний

1. Включите: `NG Auth → Настройки → Рассылка напоминаний`
2. Настройте задержку и размер батча
3. Cron-задача `ng_auth_reminder_cron` запускается автоматически

## Требования к хостингу

- PHP 8.0+ с расширениями: `json`, `mbstring`, `openssl`
- Для SMS-провайдеров: доступ к внешним API (порт 443)
- Для ЕСИА: HTTPS-сертификат, доступ к серверам Госуслуг
