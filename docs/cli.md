# WP-CLI команды

## Установка

Команды доступны автоматически при активном плагине.

## Команды

### Массовая пометка пользователей

```bash
wp ng-auth mark-unverified
```

Помечает всех пользователей без статуса верификации как `unverified`.

### Запуск рассылки напоминаний

```bash
wp ng-auth send-reminders
```

Запускает один батч рассылки напоминаний неподтверждённым пользователям.

### Статус пользователя

```bash
wp ng-auth status <user_id>
wp ng-auth status 42
```

Выводит статус, провайдера и дату подтверждения.

### Ручное подтверждение

```bash
wp ng-auth verify <user_id> [--provider=<id>]
wp ng-auth verify 42 --provider=smsaero
```

Администратор вручную подтверждает пользователя.

### Статистика

```bash
wp ng-auth stats
```

Выводит общую статистику: всего / подтверждено / не подтверждено / заблокировано.
