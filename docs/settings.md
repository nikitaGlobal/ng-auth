# Настройки NG Авторизация

## Общие настройки

- **Включить обязательное подтверждение** — глобальный переключатель.
- **Роли с обязательным подтверждением** — выбор ролей. По умолчанию все. Администратор не требует подтверждения.
- **Текст уведомления на форме регистрации** — сообщение для пользователей перед регистрацией.
- **Требовать подтверждение только для IP из РФ** — гео-IP фильтр. Если IP не российский — подтверждение не требуется.
- **Гео-IP провайдер** — выбор сервиса геолокации: НОЦ (noc.gov.ru) или Sweb (sweb.ru).
- **Режим для новых пользователей:**
  - `Ограниченный доступ` — пользователь создаётся, доступ до подтверждения ограничен
  - `Без доступа` — пользователь не создаётся до подтверждения

## OTP-настройки

- **Длина кода** — 4 или 6 цифр
- **Время жизни кода** — секунды (по умолчанию 300 = 5 минут)
- **Максимальное число попыток** — после исчерпания — блокировка
- **Интервал повторной отправки** — минимальное время между запросами кода

## Шаблоны сообщений

- `{code}` — код подтверждения (SMS OTP)
- `{site_name}` — название сайта
- `{user_name}` — имя пользователя
- `{verify_url}` — ссылка на страницу верификации
- `{user_email}` — email пользователя

## Рассылка напоминаний

- **Включить рассылку** — активирует cron
- **Задержка первого напоминания** — дни после регистрации
- **Размер батча** — пользователей за проход cron (10–200)

## Провайдеры

### SMS Aero
- **API-ключ** — в формате `email:api_key`
- **Режим тестирования** — SMS не отправляется, код записывается в Журнал верификации (действие `test_otp`)

### МТС Exolve
- Стандартные поля: включение, API Key, API Secret, endpoint

### ЕСИА (Госуслуги)
- Экспериментальная возможность
- Требует зарегистрированной мнемоники в СМЭВ
